## 2. 完整战斗流程 (14步)

根据 kcwikicn 研究，标准战斗流程如下：

```
1. 阵形选择 (陣形選択)
2. 索敌 (索敵)
3. 喷式强袭 (噴式強襲) — 基地航空队
4. 基地航空队 (基地航空隊)
5. 航空战 (航空戦)
6. 支援舰队 (支援艦隊)
7. 先制对潜 (先制対潜)
8. 开幕雷击 (開幕雷撃)
9. 交战形态 (交戦形態) — T字战法判定
10. 炮击战 (砲撃戦)
11. 雷击战 (雷撃戦)
12. 追击选择 (追击選択)
13. 夜战 (夜戦) — 如选择进入
14. 战斗结算
```

## 3. 伤害计算公式 (核心)

### 3.1 伤害上限值 (Cap Values) — 2021年更新

| 攻击类型 | 上限值 |
|---------|-------|
| 昼战炮击 | **220** |
| 昼战雷击 | **180** |
| 夜战 | **360** |
| 航空战/对潜/支援 | **170** |
| 基地航空队 | **220** |

### 3.2 伤害公式 (基本)

```
最终伤害 = ⌊(最终攻击力 - 防御力) × 弹药补正⌋
```

### 3.3 Cap 后攻击力计算

```
Cap后攻击力 = Cap值 + √(Cap前攻击力 - Cap值) 的整数部分
```

### 3.4 攻击前Cap值计算

```
Cap前攻击力 = 基本攻击力 × Cap前补正 + 轻巡轻炮补正 + 伊重巡适格炮补正
```

### 3.5 基本攻击力公式 (按舰种)

**非空母炮击:**
```
基本攻击力 = 火力 + 装备火力 + 装备加成 + 改修强化值 + 5
```

**空母炮击:**
```
基本攻击力 = [火力 + 雷装×1.3 + 爆装×1.3 + ...] × 1.5 + 55
```

**雷击:**
```
基本攻击力 = 素雷装 + 装备雷装 + 装备加成 + 改修 + 5
```

**对潜:**
```
基本攻击力 = 2×√(素对潜值) + 1.5×√(装备对潜值) + シナジー + 阵形补正
```

**夜战:**
```
基本攻击力 = 火力 + 雷装 + 改修 + 夜侦常数
※ 无阵形/交战形态修正
```

## 5. 对潜攻击 (対潜攻撃)

### 5.1 先制对潜发动条件

根据舰种和装备组合：
- **驱逐舰 (DD):** 需 声响+depth charge 或 特定装备组合
- **轻巡洋舰 (CL):** 等级 > 10 + 声响
- **航空巡洋舰 (AV):** 等级 > 10 + 声响
- **轻型航母 (CVL):** 需特定装备

### 5.2 对潜装备シナジー (协同加成)

| 装备组合 | 倍率 |
|---------|------|
| 小型声响 + 投射机 + 爆雷 | **1.4375×** |
| 大型声响 + 投射机 + 爆雷 | **1.265×** |
| 声响 + 爆雷 或 投射机 + 爆雷 | **1.15×** |
| 同类装备重复 | **不叠加** |

### 5.3 装备对潜减甲公式

```
减甲值 = Σ√(装备对潜值 - 2)
```

## 12. PT小鬼群 (PT Boat Clusters)

### 12.1 PT小鬼群火力公式

PT boat clusters use a **modified firepower formula**:

```
火力 = (阈值后火力 × 0.3 + √阈值后火力 + 10) × 对PT装备补正
```

- **阈值 (Threshold):** 10
- **夜战额外惩罚:** ×0.6
- **鱼雷攻击:** 无装备补正
- **航空战:** 随机倍率 0.5 或 0.8

### 12.2 PT装备补正

| 装备类型 | 补正倍率 |
|---------|---------|
| 小口径主炮 (2×) | **2.1×** |
| 小口径主炮 (1×) | **1.5×** |
| 副炮 | **1.3×** |
| 舰爆/水爆 | **1.4×** |
| 机枪 | **1.2×** |
| 舰攻 | **1.25×** |
| 舰战 | **1.1×** |

### 12.3 PT小鬼群命中公式

```
命中 = 基础命中 + 火力×0.2 + 装备命中补正 + 天雾改二补正
```

**天雾改二/丁 特殊补正:**
- 自身: **+64** 命中
- 相邻驱逐舰/防空舰: **+32** 命中

### 12.4 舰型命中补正

| 舰种 | 补正 |
|------|-------|
| DD/DE | 1.0 |
| CL/CT/CLT | 0.82 |
| CA/CAV/BB/BBV | 0.7 |

### 13.1 联合舰队中攻击顺序

```
步骤2: 喷式强袭 (Jet Assault)

### 13.2 基地航空队 Cap 值

- **Cap 值: 220** (不同于 standard 170)

### 13.3 基地航空队航空战公式

基地航空队使用航空战公式，但 threshold 为 **220**:

```
基地航空队攻击力 = Cap值(220) + √(攻击方战力 - Cap值)

## 15. Phase 3 研究成果 (已完成)

### 15.1 夜战 CI 发动率 (wikiwiki 夜戦)

**CI发动率公式:**
```
Luck < 50: CI項 = 15 + 運 + floor(0.75 × √Lv) + 補正
Luck ≥ 50: CI項 = 65 + √(運-50) + floor(0.8 × √Lv) + 補正
发动率(%) = (CI項 / 種別係数) × 100
```

**種別係数 (CI Type Coefficients):**
| CI类型 | 係数 |
|--------|------|
| 魚雷CI | 122 |
| 主魚CI | 115 |
| 主砲CI | 140 |
| 主副CI | 130 |

**補正 (Modifiers):**
| 条件 | 補正 |
|------|------|
| 旗舰 | +15 |
| 中破 | +18 |
| 自身探照灯 | +7 |
| 自身照明弾 | +4 |
| 敌方探照灯 | −5 |
| 敌方照明弾 | −10 |
| 熟练瞭望员 | +5 |
| 水雷瞭望员 (DD/CL/CLT) | +8 |

**DD CI优先顺序** (Lv80+ 可用2连击型):
主魚電 > 魚見電 > 魚魚水 > 魚ド水 > 魚魚

**D炮加成:** 1.25× per D型改二, 1.4× for 2×, 改三 additionally 1.05/1.1×

### 15.2 空母夜战攻击 (CV Night Attack)

**搭载補正公式:**
```
搭載補正 = A×搭載数 + B×(火力+雷装+爆装+対潜)×√搭載数 + √★
```

| 攻击类型 | A | B |
|----------|---|---|
| 夜戦/夜攻/夜爆 | 3.0 | 0.45 |
| SF系/光電管彗星/爆戦岩井 | 0 | 0.3 |

**夜戦/夜攻/夜爆 Cap:** 360 (超出使用 √ 公式)

**夜间触接补正:** +5/+7/+9 (随侦察机类型)

**日夜转换:** 夜战需要 航空優勢 或 better for night contact

### 15.3 对潜装备完整分类 (wikiwiki 対潜攻撃)

**声响 (Sonars):**
- 被动式: 九三式, 四式, 零式
- 主动式: 三式探信儀, Type124/144/147 ASDIC, HF/DF

**爆雷 (Depth Charges):**
- 九五式爆雷: √2 ≈ 1.4
- 二式爆雷: √5 ≈ 2.2

**投射机 (Depth Charge Projectors):**
- Hedgehog: √16 = 4.0
- 対潜短魚雷: √18 ≈ 4.2
- RUR-4A改: √15 ≈ 3.8
- Mk.32: √17 ≈ 4.1

**航空机 (Aircraft):** 艦攻/艦爆 (with ASW stat), 水上爆撃機, 回転翼機, 対潜哨戒機, 大型飛行艇

**迫击炮 (ASW Mortars):** 二式12cm迫撃砲改 — **无协同**

### 15.4 OASW 发动条件 (详细)

**基础条件:**
- 必须是昼战
- 联合舰队时为第二舰队
- DD/CL/CT/CLT/AO: ASW ≥ 100 + 声响
- DE: ASW ≥ 60 + 声响, 或 raw equip ASW ≥ 4
- Special DDs (Isuzu K2, Tatsuta K2, Yuubari K2, J-class, Samuel B.Roberts Mk.II, Fletcher-class): **无条件**

**CVL/CVE:** various ASW thresholds (50/65/100) with specific equipment combos
